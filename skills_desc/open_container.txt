open_container: Call this tool API to control the robot. What is the open_container API useful for? Robot will open the container at the robot's current location in order to 1. find if the target object is in the container,or 2. place the target object inside the container Parameters: [{"name": "container_name", "type": "string", "description": "the name of container the robot should open", "required": true}] Format the arguments as a JSON object.
